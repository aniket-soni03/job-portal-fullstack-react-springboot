# 💼 Job Portal — Frontend

## 📌 Description
Frontend application for the job portal built with React providing user interface for authentication, job browsing, and job application.

## 🚀 Tech Stack
- React.js
- React Router DOM
- React Hook Form
- Tailwind CSS
- Axios

## ✨ Features
- User registration and login interface
- Role-based UI for Employer and Candidate
- Job listing, search, and filter functionality
- Job application interface
- Form validation using React Hook Form
- Responsive UI for all screen sizes

## 🔗 Integration
- Connected with Spring Boot backend APIs
- Handles JWT-based authentication
- API communication using Axios

## ⚙️ Run Locally

```bash
cd frontend
npm install
npm run dev
